# Adwield — Landing Page

A clean, deploy-ready static site (vanilla `index.html` + `styles.css` + `script.js`, no build step, no dependencies) for the Adwield ad-creative engine — dark navy / gold / cyan "Command Deck" aesthetic, responsive 360px → 1440px+, accessible (semantic HTML, focus states, `prefers-reduced-motion`), with tasteful scroll-reveal, count-up, and a subtle pointer-tracked hero accent. Before going live, replace the literal merge fields `{{first_strike_form}}` and `{{book_call}}` with real URLs and the `[ CASE STUDY PLACEHOLDER ]` block with a documented result. Deploy by dropping this folder onto any static host — **Vercel** (`vercel` / `vercel --prod` from this directory), **Netlify** (drag-and-drop the folder at app.netlify.com/drop, or `netlify deploy --prod`), or **Cloudflare Pages** (`wrangler pages deploy .`, or connect the repo with no build command and output dir `.`).
